# Yu-Gi-Oh-Duel-Links
## 遊戲王決鬥聯盟(Yu-Gi-Oh! Duel Links) 自動掛機腳本 (DLAUTOSC)

## 刷角色經驗、等級、素材、鑰匙、傳送門(LV10)、決鬥者(路人、傳奇、其他、活動)

## 使用Steam開啟Yu-Gi-Oh-Duel-Links後執行腳本

## 下載檔案後解壓縮雙擊執行檔即可執行腳本

## 模擬滑鼠移動點擊，非外掛腳本、不修改遊戲數值。
